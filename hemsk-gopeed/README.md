# Gopeed for Umbrel

Gopeed is a high-speed, open-source download manager built with Golang and Flutter. It supports HTTP, HTTPS, BitTorrent, and Magnet protocols.

## Credits

This app is an Umbrel-compatible version of **[Gopeed](https://gopeed.com/)**. All credit for the development and maintenance of Gopeed goes to the **Gopeed** project and its team at GopeedLab.

## Features

- **High Speed:** Leverages Golang's concurrency for maximum performance.
- **Cross-Protocol:** Supports HTTP, HTTPS, BitTorrent, and Magnet links.
- **Modern UI:** Built with Flutter for a sleek, responsive experience.
- **Extensible:** Supports JavaScript-based extensions and an Open API.
